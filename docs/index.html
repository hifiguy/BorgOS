Architecture

Four-layer memory plane

L1

Working — Agent-Local Scratchpad

Fast, session-scoped, per-agent. Promotes to L2 via session-end consolidation. Maintained by Dream Cycle batch operations.

L2

Semantic — Shared Fleet Recall

Qdrant with hybrid BM25 + dense vectors. Per-agent API keys. Confidence-weighted ranking. All writes through validated endpoint only — never direct.

L3

Provenance — Forensic Record

Append-only PostgreSQL with bitemporal columns. No UPDATE, no DELETE. Every memory fact is a permanent forensic record with full provenance chain.

L4

Reference — Ground Truth

Operator-curated knowledge. Kernel-enforced read-only. Confidence 1.0. Agents cannot modify — only propose changes for human review.

Trust ascends: L1 (agent-local) → L2 (shared fleet) → L3 (forensic) → L4 (operator ground truth). L4 always wins on collision.
