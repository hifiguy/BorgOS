The Problem

Every team running multiple AI agents hits the same wall: agents don't share what they learn. When they do, there's nothing stopping them from creating echo chambers where guesses get amplified into facts.

Agent A writes a guess. Agent B retrieves it, writes it back as a "corroborating" fact. Now two agents agree. Confidence goes up. The wrong answer looks more credible than the right one.

This isn't theoretical. HaluMem (2025) proved write-time extraction compounds hallucinations. MemoryGraft (2025) demonstrated persistent compromise through poisoned experience. And that was single-agent. Multiply it across a fleet.

Mem0 ($24M raised, 186M API calls/month) has zero write validation. Zep (Samsung, AWS) has no write pipeline. Letta ($10M) has access control but no validation. The entire market has the same gap.

Four-Layer Architecture

L4: Reference — Ground Truth
Operator-curated, kernel-enforced read-only. Agents cannot modify. Confidence: 1.0
L1: Provenance — Forensic Record
Append-only PostgreSQL, bitemporal. No UPDATE, no DELETE. Permanent audit trail.
L2: Semantic — Shared Recall
Qdrant, cross-agent, confidence-weighted ranking. Per-agent API keys.
L3: Working — Agent Scratchpad
Fast, ephemeral, per-agent. Promotes to L2 via validated pipeline.

When reference knowledge and agent knowledge collide on the same topic, reference wins. Always. The agent version gets a retrieval penalty.

Confidence & Anti-Laundering

Three Confidence Tiers

  • Hypothesis — any agent can claim. Self-referential content capped here.
  • Experimental — requires corroboration from a different agent.
  • Verified — requires operator confirmation or multi-agent consensus.

Echo Chamber Detection

BorgOS traces corroboration ancestry 3 hops back. If the "corroborating" agent's knowledge traces back to the original claim, it's flagged as laundering. Three checks: shared ancestry tracing, high-similarity ancestor detection, and time-based suspicion for rapid round-trip patterns.

10 Anomaly Detectors

Volume spikes, confidence manipulation, semantic drift, self-referential floods, corroboration laundering, content-type abuse, rapid-fire writes, identity injection, migration mismatch, merge storms. Weighted composite scoring with four response tiers.
